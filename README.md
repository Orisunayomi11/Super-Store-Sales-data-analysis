# Super-Store-Sales-data-analysis
Power BI dashboard analyzing monthly sales trend and profit generated from each category
The West Region leads in both sales and profit.
- Technology category contributes the highest profit margin.
- Consumer segment generates the most revenue, but Corporate has higher profit per order.
